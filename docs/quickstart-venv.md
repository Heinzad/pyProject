Quickstart the Virtual Environment
==================================

*Bash terminal:*

```bash
source .venv/Scripts/activate
```
```bash
deactivate
```

*PowerShell terminal:*

```powershell
Set-ExecutionPolicy Unrestricted -Scope Process
```
```powershell
.venv\Scripts\Activate.ps1
```
```powershell
deactivate
```
